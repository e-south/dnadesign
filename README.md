## Overview

`dnadesign` is a collection of modular bioinformatic pipelines and helper packages related to biological sequence design.

## Available tools

- [**usr**](src/dnadesign/usr): Universal Sequence Record utilities for inspecting datasets and Parquet files.
- [**notify**](src/dnadesign/notify): Tool-agnostic webhook notifier for batch runs (Slack, Discord, generic webhooks).
- [**densegen**](src/dnadesign/densegen): DNA sequence design pipeline built on the [`dense-arrays`](https://github.com/e-south/dense-arrays) framework.
- [**infer**](src/dnadesign/infer): Model-agnostic wrapper for DNA/protein language models such as [Evo2](https://github.com/ArcInstitute/evo2/tree/main).
- [**opal**](src/dnadesign/opal): [EVOLVEpro-style](https://www.science.org/doi/10.1126/science.adr6006) active-learning tool for DNA/protein sequence design campaigns.
- [**cluster**](src/dnadesign/cluster): Parquet/CSV-first toolkit for clustering, UMAP visualization, and related analyses.
- [**billboard**](src/dnadesign): Quantifies regulatory diversity of dense-array DNA libraries generated by `densegen`.
- [**libshuffle**](src/dnadesign/libshuffle): Iterative subsampling workflow that uses `billboard` as its analysis engine.
- [**nmf**](src/dnadesign/nmf): Applies NMF to sequence libraries to identify higher-order TFBS combinations.
- [**latdna**](src/dnadesign/latdna): Pipeline for latent-space analysis of DNA sequences.
- [**cruncher**](src/dnadesign/cruncher): PWM-driven sequence optimization pipeline with pluggable parsers and optimizers.
- [**tfkdanalysis**](src/dnadesign/tfkdanalysis): Pipeline for TFKD analysis with PPTP-seq context ([Han et al., 2023](https://doi.org/10.1038/s41467-023-41572-4)).
- [**aligner**](src/dnadesign/aligner): Wrapper around Biopython `PairwiseAligner` for Needleman-Wunsch-style global alignment scoring.
- [**permuter**](src/dnadesign/permuter): Pipeline for biological sequence permutation and downstream evaluation.
